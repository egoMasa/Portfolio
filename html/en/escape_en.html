ESCAPE GAME

Presentation of the project

The goal of this project is to create an escape game consisting of 5 events

For this I have set up a system of different transmissions by each of them as :

  • - Infra-red
  • - Radio waves
  • - BLE
  • - Telephony

A web site is set up to play and for the administrators to control the good progress of the teams
